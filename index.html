Panduan Siswa (Download & Instal)

  1. Klik tombol Download APK Sekarang di atas langsung dari HP Android kamu.
  2. Setelah selesai mengunduh, buka file APK untuk menginstal.
  3. Jika sistem memblokir, masuk ke Setelan / Settings lalu aktifkan opsi Izinkan dari sumber ini (Allow from this source).
  4. Buka aplikasi Exambro setelah berhasil terpasang.
  5. Wajib: Berikan izin akses kamera saat aplikasi meminta agar scanner QR berfungsi.
  6. Saat dialog sistem meminta mode kunci layar (Screen Pinning), kamu WAJIB klik OKE / IZINKAN. Menolak mode kunci akan memicu alarm pelanggaran otomatis.

Panduan Guru (Custom Password)

Sandi darurat bawaan untuk keluar dari aplikasi adalah guru123. Guru/Panitia dapat mengatur sandi penutup yang unik untuk tiap sesi ujian langsung melalui tautan soal.

Cara Modifikasi Tautan Ujian:

Sisipkan teks ?pwd=sandi_pilihan_anda tepat di ujung link Google Forms atau web ujian sebelum diubah menjadi QR Code.

https://docs.google.com/forms/.../viewform?pwd=ujianselesai
  • Jika siswa memindai QR hasil tautan di atas, sandi keluar otomatis berubah menjadi: ujianselesai.
  • Sifat Sandi: Bersifat sensitif terhadap huruf besar/kecil (*case-sensitive*).
  • Gunakan tautan penuh yang sudah dimodifikasi ke platform pembuat QR Code gratis pilihan Anda, lalu cetak untuk diberikan kepada siswa di ruang ujian.
