Panduan Siswa

  1. Klik tombol Download APK di atas langsung dari HP Android kamu.
  2. Setelah selesai mengunduh, buka file APK untuk menginstal.
  3. Jika sistem memblokir, masuk ke Setelan / Settings lalu aktifkan opsi Izinkan dari sumber ini (Allow from this source).
  4. Buka aplikasi Exambro setelah berhasil terpasang.
  5. Wajib: Berikan izin akses kamera saat aplikasi meminta agar scanner QR berfungsi.
  6. Saat dialog sistem meminta mode kunci layar (Screen Pinning), kamu WAJIB klik OKE / IZINKAN. Menolak mode kunci akan memicu alarm pelanggaran.

Panduan Guru

Sandi darurat bawaan untuk keluar dari aplikasi adalah guru123. Panitia dapat mengatur sandi penutup unik untuk tiap sesi ujian.

Cara Modifikasi Tautan Ujian:

Sisipkan teks ?pwd=sandi_pilihan tepat di ujung link Google Forms/web ujian sebelum diubah menjadi QR Code.

https://docs.google.com/forms/.../viewform?pwd=ujianselesai
  • Jika siswa memindai QR hasil tautan di atas, sandi keluar otomatis berubah menjadi: ujianselesai.
  • Sifat Sandi: Sensitif terhadap huruf besar/kecil.
  • Gunakan tautan penuh yang dimodifikasi tersebut ke platform pembuat QR Code gratis.
